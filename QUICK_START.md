## 📋 5分钟快速教程

### 第1步: 打开应用 (30秒)

1. 访问URL: https://3000-itlgpys4bhtsrug4lqi7i-02b9cc79.sandbox.novita.ai
2. 你会看到三栏布局:
   - **左侧**: 项目列表和图像列表
   - **中间**: 图像查看器和标注工具
   - **右侧**: 标注列表和统计信息

### 第2步: 选择项目和图像 (30秒)

1. 在左侧 **"项目列表"** 中，点击 **"肺癌组织分析"**
2. 项目下方会显示该项目的图像
3. 点击 **"肺癌切片-01.jpg"**
4. 中间画布区域会显示图像（示例图像）
5. 右侧会显示已有的标注

### 第3步: 添加点标注 (1分钟)

1. 点击工具栏中的 **圆圈图标** 🔵 (点标注工具)
2. 在 **"标签"** 下拉菜单中选择 **"淋巴细胞"**
3. 在图像上随意点击几次
4. 每次点击都会创建一个蓝色的点标注
5. 右侧的标注列表和统计会实时更新

### 第4步: 添加多边形标注 (2分钟)

1. 点击工具栏中的 **多边形图标** 🟣
2. 标签选择 **"肿瘤细胞"**
3. 在图像上连续点击4-5次，勾画一个多边形
4. 最后点击接近起点的位置（会自动闭合）
5. 创建的多边形会显示为红色半透明区域
6. 右侧会显示该标注的面积

### 第5步: 视图控制 (1分钟)

**缩放图像**:
- 使用鼠标滚轮向上滚动放大
- 向下滚动缩小
- 或点击工具栏的 🔍+ 和 🔍- 按钮

**移动视图**:
- 点击 **手掌图标** ✋
- 按住鼠标左键拖拽移动
- 或直接在滚动时按住鼠标拖拽

**重置视图**:
- 点击 **⛶ 图标** 恢复默认视图

### 第6步: 查看统计和导出 (30秒)

1. 右侧 **"统计信息"** 面板显示:
   - 总标注数量
   - 各类细胞的数量分布
   - 颜色编码对应标签

2. 点击 **"导出数据"** 按钮
3. 浏览器会下载一个JSON文件
4. 文件包含所有标注的完整信息

## 💡 实用技巧

### 技巧1: 快速点标注
适合大量细胞计数，不需要精确轮廓时使用。

**步骤**:
1. 选择点标注工具
2. 选择细胞类型
3. 在每个细胞中心快速点击

### 技巧2: 精确多边形
需要测量细胞面积或分析形态时使用。

**步骤**:
1. 先放大图像到合适倍数
2. 选择多边形工具
3. 沿细胞边缘仔细点击
4. 顶点越多，轮廓越精确

### 技巧3: 批量操作
通过API批量导入AI模型的预测结果。

**示例**:
```bash
curl -X POST http://localhost:3000/api/images/1/annotations/batch \
  -H "Content-Type: application/json" \
  -d '{
    "annotations": [
      {"annotation_type":"point","label":"lymphocyte","coordinates":[{"x":100,"y":200}]},
      {"annotation_type":"point","label":"tumor","coordinates":[{"x":300,"y":400}]}
    ]
  }'
```

## 🔧 常见问题快速解决

### Q: 标注后看不见？
**A**: 点击重置视图按钮 ⛶

### Q: 工具不响应？
**A**: 确认已选择图像（绿色高亮）

### Q: 多边形画错了？
**A**: 切换到删除工具 🗑️，点击多边形删除

### Q: 图像太小/太大？
**A**: 使用鼠标滚轮调整缩放
